##  О проекте

Проект решает задачу поиска аниме по очень специфическим критериям из базы **20,000+ аниме** с сайта [shikimori.one](https://shikimori.one).
На сайте почти у каждого аниме присутстуют: жанр, тема, описание. По ним и делалась фильтрация.

### Целевые критерии:
- Главная героиня — **взрослая девушка** (18+ лет)
- Сюжет **сфокусирован на романтике** и развитии отношений.
- События происходят в **реальном мире** (без магии/мистики)
- Отсутствуют элементы **насилия и жестокости**



### ❓ Зачем нужен проект?

Простой поиск по жанру "Романтика" на сайте не дал нужных результатов по двум причинам:
1. **Жанр "Романтика" часто второстепенен.** Даже если он указан, основной сюжет может строиться на спорте, учёбе, работе или хобби героев, а романтика идёт фоном. Мне же была важна именно **романтика как основа сюжета**.
2. **Жанры и темы не всегда точны.** Элементы мистики, магии или жестокости могут присутствовать в описании аниме, даже если не указаны в метаданных явно.

**Примеры:**
- *"Волчица и семеро друзей"* — жанры безобидные, но описание намекает на опасность и жестокость
- *"Тёко, сестрёнка"* — есть мистика в сюжете, хотя в жанрах не указана

Поэтому был создан pipeline с AI-анализом описаний для точной фильтрации.



### Результат:
**20,771 аниме** → **12 идеальных совпадений** (reduction rate: 99.94%)

## Полный pipeline (с нуля)

Если хотите запустить весь процесс от парсинга до результата:

```bash
cd src

# Этап 1: Парсинг (~10-15 часов для всех аниме)
python 1_parse_anime.py

# Этап 2: Базовая фильтрация (TV сериалы, рейтинг ≥6.0)
python 2_filter_basic.py

# Этап 3: Фильтрация романтики (жанры, темы)
python 3_filter_romantic.py

# Этап 4: AI-анализ (~5-10 минут, ~$0.10-0.20)
python 4_analyze_with_ai.py

# Этап 5: Финальная фильтрация
python 5_final_filter.py

# Просмотр результатов
python view_results.py
```

## Как работает pipeline

### Этап 1: Парсинг данных
- **Входные данные:** shikimori.one
- **Выходные данные:** `data/raw/anime_database.json` (20,771 аниме)
- **Что извлекается:** названия, описания, жанры, рейтинги, эпизоды

### Этап 2: Базовая фильтрация
- **Фильтры:**
  - ✅ Только TV сериалы
  - ✅ Есть описание
  - ❌ Исключены продолжения/сиквелы
  - ❌ Исключён рейтинг G (детский)
  - ⭐ Зрительский рейтинг ≥ 6.0
- **Результат:** 2,089 аниме

### Этап 3: Фильтрация романтики
- **Фильтры:**
  - ✅ Жанр: Романтика
  - ❌ Исключена тема: Школа
  - ❌ Исключены жанры: Сверхъестественное, Фэнтези, Sci-Fi, Махо-сёдзё
- **Результат:** 117 аниме

### Этап 4: AI-анализ описаний
- **Модель:** OpenAI GPT-4o-mini
- **Анализируется:**
  -  Пол главного героя
  -  Наличие насилия
  -  Наличие мистики/магии
  -  Романтика как основа сюжета
  -  Возраст главного героя

### Этап 5: Финальный отбор
- **Критерии:**
  - ✅ hero = "female"
  - ✅ violence = "нет"
  - ✅ mystical = "нет"
  - ✅ love_vibes = "да"
  - ✅ approximateage ≥ 18
- **Результат:** 12 аниме ✨
